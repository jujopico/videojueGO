# Node / Create-React-App mashup

## Setup

* `git clone` this repo
* `cd` into it.
* `yarn install`
* `cd client && yarn install`

## To make it run

* `yarn start` from project root
* `yarn start` inside of `/client`

## To deploy

* `heroku create your-app-name`
* `git push heroku master`
